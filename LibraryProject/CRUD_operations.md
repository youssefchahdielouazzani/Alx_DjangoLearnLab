# CRUD Operations

This document summarizes the CRUD operations performed on the Book model
using the Django shell.

## Create
A Book instance was created with the title "1984", author "George Orwell",
and publication year 1949.

## Retrieve
The Book instance was retrieved and its attributes (title, author,
publication year) were displayed.

## Update
The title of the Book was updated from "1984" to "Nineteen Eighty-Four".

## Delete
The Book instance was deleted and the database was confirmed to be empty.
